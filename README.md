# Sales Performance Analysis

## About This Project
This project analyzes retail sales data to find business insights 
using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts and Visualizations

## What I Analyzed
- Sales performance by product
- Sales by region (Vancouver vs Toronto)
- Monthly sales trends
- High, Medium, Low performance categories

## Skills Demonstrated
- Data cleaning
- Pivot Tables
- IF formulas
- Data visualization
- Business insights
